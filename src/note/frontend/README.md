## 初识Web前端

- 什么是Web

  - 全球广域网，也称万维网（World Wide Web），能够通过浏览器访问的网站。

- Web网站开发模式

   ![Web网站开发模式](https://gitee.com/yindong-wen/mypicgo_img/raw/master/image/20260302103110487.png)

- 网页本质：前端代码

- 前端代码如何转换成网页

  - 通过浏览器转化（解析和渲染）成用户看到的网页

  - 浏览器中对代码进行解析渲染的部分，称为浏览器内核

  - 不同浏览器的内核，对应前端代码的解析效果存在差异

- Web标准：也称网页标准，由一系列标准组成，大部分由W3C（World Wide Web Consortium，万维网联盟）负责制定。包含以下三个部分：

  - HTML：负责网页的结构（页面元素和内容）。见[HTML](../language/html.md)。

  - CSS：负责网页的表现（页面元素的外观、位置等页面样式，如：颜色、大小等）。见[CSS](../language/css.md)。

  - JavaScript：负责网页的行为（交互效果）。见[JavaScript](../language/javascript.md)。
