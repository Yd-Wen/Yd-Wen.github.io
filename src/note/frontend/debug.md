1. 打开浏览器开发者工具

- 按下快捷键 F12

- 或鼠标右键元素点击“检查”

2. 查看元素信息

- 元素检查器（点击元素，查看元素属性）

- 样式查看器（查看元素样式）

- 控制台（查看运行错误）

3. 调试元素样式

- 选中元素，查看元素样式（在样式查看器中查看）

- 调试元素样式：

  - 直接修改样式查看效果

  - 样式属性多选框

::: tip 细节
- 如果是错误属性，有黄色叹号的警告标志。

- 样式属性多选框：在样式查看器中勾选样式属性，查看效果。勾选后属性生效，否则不生效。
:::
