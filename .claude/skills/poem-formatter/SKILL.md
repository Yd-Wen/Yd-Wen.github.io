# 诗歌格式化技能

接收诗歌文本，按照规范格式化后保存到 `src/piece/poem/` 目录。然后在索引文件 `src/piece/poem/README.md` 中添加索引。

## 输入格式要求

输入文本应包含以下信息（用换行或分隔符区分）：
- 标题：诗歌标题
- 作者：作者姓名
- 时间：日期（用于生成文件名，格式 yyyymmdd）
- 内容：诗歌正文
- 注解：可选，诗歌注释说明

## 格式化规则

1. **标题**：居中加粗

2. **作者**：另起一行居中显示

3. **诗歌内容**：
   - **词格式**：两个自然段，每段段首空两个字符（使用 `&emsp;&emsp;` 或两个全角空格）
   - **律诗格式**：多行，每行居中，每行一句，10或14个字

4. **注解**：以 "【注】" 开头，每个注解以阿拉伯数字开头（如 1. 2. 3.）

5. **Frontmatter**：文件顶部添加 YAML frontmatter

6. **居中**：使用 `::: center` 标记，`:::`结束

## 索引文件格式要求
- 索引文件名为 `README.md`
- 索引文件格式为：
```markdown
## [诗歌标题](文件名)
```

## 输出文件格式示例

```markdown
---
title: 衡阳与梦得分路赠别
icon: poem
date: 2019-09-13
category: 随笔
tag:
  - 诗歌
---

::: center
**衡阳与梦得分路赠别**

柳宗元

十年憔悴到秦京，谁料翻为岭外行。

伏波故道风烟在，翁仲遗墟草树平。

直以慵疏招物议，休将文字占时名。

今朝不用临河别，垂泪千行便濯缨。
:::

**【注】**
1. 柳宗元、刘禹锡因参与"永贞革新"而久遭贬谪，奉旨返京后再被贬谪。此诗作于二人在衡阳分手、各赴谪地之时。
2. 伏波：东汉伏波将军马援。
3. 翁仲：秦时巨人，秦始皇曾铸金人以像翁仲，后世称石像或墓道石为翁仲。
4. 慵疏：懒散粗疏，这是托词，其实是说不愿与腐朽势力同流合污。
5. 濯：洗；缨：系冠的丝带。濯缨，洗缨，《楚辞·渔父》有"沧浪之水清兮，可以濯我缨"，后以濯缨喻超脱世俗。

## 输出索引示例

```markdown
### [衡阳衡阳与梦得分路赠别](20190913.md)
```
```

## 执行步骤

1. 解析输入文本，提取标题、作者、时间、内容、注解
2. 判断诗歌类型（词/律诗）：
   - 如果内容明显分为两段且每段较长 → 按"词"格式处理
   - 如果内容多行且每行较短（10-14字）→ 按"律诗"格式处理
3. 按规则格式化诗歌内容
4. 构建文件路径：`src/piece/poem/<时间>.md`
5. 检查目标目录是否存在，不存在则创建
6. 写入格式化后的内容到文件
7. 在索引文件 `src/piece/poem/README.md` 中添加索引
8. 返回保存的文件路径和操作结果

## 注意事项

- 时间格式应为 `yyyymmdd`（如 20240329），用于生成文件名
- 文件中的日期 frontmatter 格式为 `yyyy-mm-dd`
- 如果目标文件已存在，询问用户是否覆盖
- 注解部分可选，如果没有注解则省略【注】部分
